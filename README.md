# Training Planner

Графическое приложение для планирования тренировок. Позволяет добавлять, удалять и фильтровать тренировки по типу и дате. Все данные сохраняются в файл `trainings.json`.

**Автор:** Михаил Колокольников

## Возможности

- Добавление тренировки с указанием даты, типа и длительности.
- Отображение списка тренировок в удобной таблице.
- Фильтрация по типу тренировки (Бег, Плавание, Силовая и т.д.) и конкретной дате.
- Удаление выбранной тренировки.
- Автоматическое сохранение данных в JSON-файл.
- Валидация ввода: дата в формате YYYY-MM-DD, длительность — положительное целое число.

## Требования

- Python 3.6 или выше
- Библиотека `tkinter` (входит в стандартную поставку Python)
